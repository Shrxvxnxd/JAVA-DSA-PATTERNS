# Star & Character Patterns

A collection of 14 standard Data Structures and Algorithms (DSA) star and character pattern printing programs consolidated into a single, clean, interactive Java CLI tool: [patterns.java](patterns.java).

## How to Run CLI Tool

To compile the program:
```bash
javac patterns/patterns.java
```

To run the program:
```bash
java patterns.Main
```

Upon running, you will be prompted with a CLI menu to select a pattern (1-14) or exit (0), and then input the size parameter `n`.
