# Java DSA Patterns & Algorithms

A collection of standard Data Structures and Algorithms (DSA) patterns, array operations, and CLI printing tools implemented in Java.

This repository is organized into distinct sections:

## 📁 Sections

- **[Java Installation Guide](installation.md)**: Step-by-step instructions to download, install, and configure Java (JDK) on Windows, macOS, and Linux.
- **[Arrays Problems & Algorithms](Arrays/arrays.md)**: A collection of standard array operations, search algorithms, and math-related array utility scripts located under the `Arrays/` folder.
- **[Star & Character Patterns](patterns/patterns.md)**: Details, examples, and instructions for the 14 interactive pattern printing programs built into `patterns.java` inside the `patterns/` folder.

---

## 🛠️ How to Run

### Star & Character Patterns CLI Tool
To run the interactive CLI tool for patterns:
```bash
cd patterns
javac patterns.java
java Main
```

### Array Programs
To run any array program, compile and run the respective file from the root directory. For example, to run Binary Search:
```bash
javac Arrays/Binary_search.java
java Arrays.Main
```
*(Note: Ensure you run the commands from the repository root directory unless instructed otherwise).*
